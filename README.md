# ThreeJs GLB Walking Demo

Minimal three.js demo with a hand-rigged GLB character.
Click to move the character using simple raycasting.

## Demo
- URL: https://three-glb-walk.vercel.app/
- Video: https://x.com/HokugoW/status/2010549158307102748

## Features
- Pure CDN three.js
- Single index.html
- Hand-rigged GLB character
- Click-to-walk interaction

## Tech
- three.js via CDN
- GLB(export with blender)
- tailwindcss via CDN
